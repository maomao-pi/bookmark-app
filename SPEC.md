# 网站收藏夹 - 功能规格文档

## 1. 项目概述

- **网站名称**: 知链方舟
- **slogan**: 你的每一份收获，都有可靠归处
- **核心前端框架**: React
- **UI 组件库**: Ant Design
- **基础技术层**: 基于HTML5+CSS3+ES6+TypeScript开发，通过 TypeScript 实现类型校验，提升大型项目的代码健壮性，同时采用 Flex/Grid 实现响应式布局，适配多端设备；
- **核心功能**: 管理个人网站收藏，支持分类管理、详情查看、文章关联、主题切换等
- **目标用户**: 需要整理和快速访问常用网站的用户

## 2. UI/UX 规格

### 2.1 布局结构

- **展示模式**: 宽屏全幅展示，内容区域铺满视口宽度；
- **不采用居中固定宽度布局**（如 max-width 居中、margin auto 居中），充分利用大屏幕空间
- **Header**: 标题 + 操作按钮（深色模式切换）
- **工具栏**: 分类筛选下拉 + 添加按钮 + 搜索框
- **分类管理**: 可创建新分类、编辑、删除
- **收藏网格**: 响应式网格布局展示收藏卡片
- **空状态**: 无收藏时显示的引导界面

### 2.2 响应式断点

- **桌面**: > 1024px
- **平板**: 640px - 1024px
- **手机**: < 640px (单列网格，垂直布局)

### 2.3 视觉设计

#### 颜色方案

**浅色模式**:
- 主背景: `#F7F5F2`
- 卡片背景: `#FFFFFF`
- 标签背景: `#E8E4DF`
- 主文字: `#1A1815`
- 次要文字: `#6B6560`
- 辅助文字: `#9C9690`
- 主题强调色: `#3B82F6` (蓝色)
- 强调色背景: `#EFF6FF`
- 边框: `#E0DCD7`

**深色模式**:
- 主背景: `#0F0F0F`
- 卡片背景: `#1A1A1A`
- 标签背景: `#2A2A2A`
- 主文字: `#F0F0F0`
- 次要文字: `#A0A0A0`
- 辅助文字: `#707070`
- 强调色背景: `#1E3A5F`
- 边框: `#333333`

#### 字体
- **标题字体**: DM Serif Display (衬线体)
- **正文字体**: Outfit (无衬线体)
- **字号**:
  - 主标题: clamp(2rem, 5vw, 2.8rem)
  - 卡片标题: 1rem
  - 描述: 0.85rem
  - 标签: 0.7-0.85rem

#### 间距系统

- 容器内边距: 40px (桌面) / 24px (移动)
- 卡片间距: 18px
- 组件间距: 12-20px
- 圆角: 8px (小) / 16px (中) / 24px (大)

#### 动画效果
- 页面加载: fadeInDown / fadeInUp 动画
- 卡片悬停: translateY(-4px) + 阴影加深
- 卡片悬停顶部条纹: scaleX 动画
- 背景浮动圆点: float 动画 (6-12秒循环)
- 模态框: scale + translateY 过渡

### 2.4 组件

#### 按钮

- **主要按钮**: 蓝色背景，白色文字，圆角16px
- **次要按钮**: 透明背景，灰色边框，圆角16px
- **图标按钮**: 44x44px，正方形，圆角16px
- **按钮悬停**: 轻微上浮 + 阴影 + 颜色变化

#### 收藏卡片

- 白色/深色背景，圆角24px
- 顶部4px强调色条纹（悬停时显示）
- 左侧网站图标 (40x40px)
- 标题 + 分类标签 + 描述预览
- 右侧操作按钮（查看详情、编辑、删除）
- 点击卡片任意位置可查看详情

#### 模态框

- 居中显示，半透明黑色遮罩
- 圆角24px，白色/深色背景
- 模糊背景效果
- 表单输入框组 / 详情展示区

#### 空状态

- 图标 + 标题 + 描述 + 引导按钮
- 垂直居中显示

## 3. 功能规格

### 3.1 收藏网站管理

#### 添加收藏

- 点击"添加"按钮打开模态框
- 填写：
  - 网站标题（必填）
  - 网站网址（必填）
  - 描述（选填）
  - 分类（选填，可选择已有分类或创建新分类）
- 点击保存后添加到列表顶部

#### 查看详情

- 点击卡片打开详情弹窗
- 详情弹窗显示：
  - 网站图标、标题、完整URL
  - 网站描述
  - 所属分类
  - **关联的文章列表**（可添加/管理）
  - 创建时间
- 可从详情弹窗添加该网站下的收藏文章

#### 编辑收藏

- 点击卡片的编辑按钮打开模态框
- 预填现有数据
- 点击更新保存修改

#### 删除收藏

- 点击卡片的删除按钮
- 弹出确认对话框
- 确认后删除并更新界面

### 3.2 分类管理

#### 创建分类

- 点击分类下拉旁的"+"按钮
- 输入新分类名称
- 确认后添加到分类列表

#### 筛选分类

- 下拉选择分类进行筛选
- 选择"全部"显示所有收藏
- 支持实时搜索过滤

#### 管理分类

- 可编辑分类名称
- 可删除分类（删除时提示是否删除该分类下的所有收藏）

### 3.3 关联文章管理

#### 添加文章

- 在详情弹窗中点击"添加文章"
- 填写：
  - 文章标题（必填）
  - 文章URL（必填）
  - 文章描述（选填）
- 多个文章以列表形式展示

#### 文章列表操作

- 点击文章标题可打开链接
- 可编辑文章信息
- 可删除文章

### 3.4 搜索功能

- 实时搜索，匹配：标题、描述、URL、分类、文章标题
- 输入即搜，无延迟

### 3.5 主题切换

- 点击深色模式按钮切换
- 主题偏好保存到 localStorage
- 刷新后保持上次选择的主题

### 3.6 其他功能

#### 复制链接

- 点击复制按钮复制 URL 到剪贴板
- 显示 Toast 提示

#### 打开链接

- 点击打开按钮在新标签页打开 URL
